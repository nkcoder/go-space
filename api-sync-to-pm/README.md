# Help

```sh
API sync tool that imports OpenAPI documentation to Postman collections.

Options:
  -doc-api-key string
        The OpenAPI doc API key
  -pm-api-key string
        The Postman API key
  -pm-workspace-id string
        The Postman workspace ID

```

# How to run

```sh
go run . --doc-api-key=xxx --pm-api-key=xxx --pm-workspace-id=xxx
```
